# Hotel Management System

Проект для управления информацией о гостинице. Реализован на C++ и Python в рамках учебного задания.

## Функциональность

- Хранение информации о названии гостиницы, количестве мест, заселенности и стоимости проживания
- Заселение и выселение гостей
- Подсчет выручки за указанный период
- Свойства только для записи (название и стоимость)
